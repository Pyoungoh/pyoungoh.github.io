# 심박존 계산기

[Pace Lab 메인으로 돌아가기](/pacelab/)

나이, 최대 심박수, 안정 시 심박수, 역치 심박수를 기준으로 러닝 심박존을 계산해 보세요.

이 심박존 계산기는 Easy Run, Tempo (LT), 역치 훈련, 인터벌 훈련에 맞는 훈련 강도를 더 정확하게 설정하는 데 도움을 줍니다.

## What you can do

- 여러 방식으로 심박존 계산
- 나이 기반, 최대 심박수, 안정 시 심박수, 역치 심박수 입력 활용
- 훈련 강도별 심박존 범위 확인
- 입력값을 조정하며 결과 바로 비교

## When to use it

이럴 때 활용해 보세요:

- Easy Run과 장거리 러닝의 심박 목표를 정하고 싶을 때
- Tempo (LT), 역치 훈련, 인터벌 훈련을 계획하고 싶을 때
- 계산 방식에 따른 심박존 차이를 비교하고 싶을 때
- 더 일정한 강도로 훈련하고 싶을 때

## Example

최대 심박수가 180bpm이고 안정 시 심박수가 60bpm이라면:

- 회복 및 Easy Run 구간의 심박존은 더 낮게 설정됩니다
- Tempo (LT) 및 역치 구간의 심박존은 더 높게 설정됩니다

심박 기반 훈련 계획을 더 체계적으로 구성하는 데 활용할 수 있습니다.
