# 400mラップ計算

[Pace Lab ホームへ戻る](/pacelab/ja/)

トラック練習向けに、ランニングのペースを400mラップタイムへ、またはラップタイムをペースへ変換できます。

この400mラップ計算は、標準的な400mトラックでペースとラップタイムを相互に変換し、トラックセッションの計画や分析に役立ちます。

## What you can do

- ペース（min/km または min/mi）を400mラップタイムに変換
- ラップタイムを対応するランニングペースに変換
- 目標ラップでインターバルトレーニングを計画
- 入力を調整して結果をすぐ更新

## When to use it

こんなときに使えます:

- 一定のラップペースでトラック練習を組みたいとき
- レースペースをトラックのラップ目標に変えたいとき
- インターバルセッション（例: 400mリピート）を分析したいとき
- スピード練習でペース精度を高めたいとき

## Example

目標ペースが 5:00 min/km の場合:

- 400mラップタイム: 2:00

400mを 1:48 で走った場合:

- 換算ペース: 約 4:30 min/km

トラック練習を細かく調整しながら、ペース戦略を整えるのに便利です。
