# Pace Lab

**ランナーのための、ペース、通過タイム表、トレーニングペース、心拍ゾーン、レース予想タイム、補給戦略ツール。**

Pace Labは、トレーニングとレース計画に使える無料のランニング計算アプリです。ペースと速度の計算、通過タイム表の確認、レース予想タイム、エナジージェルの補給計画、トレーニングペースと心拍ゾーンの確認をひとつのアプリで行えます。

必要な機能にすぐアクセスして、わかりやすい結果をその場で確認できます。

## Features

### [ペース計算機](/pacelab/ja/pace-calculator/)

距離と時間から、ランニングのペースと速度を計算します。

### [レースペース計算](/pacelab/ja/race-pace-calculator/)

目標タイムと距離から目標ペースを計算し、詳細な通過タイム表を確認できます。

### [ペース・速度変換](/pacelab/ja/pace-speed-converter/)

ロード、トレッドミル、トラック練習向けに、ペースと速度を相互に変換します。

### [400mラップ計算](/pacelab/ja/400m-lap-calculator/)

トラック練習向けに、ペースを400mラップタイムへ、またはラップタイムをペースへ変換します。

### [トレーニングペース計算](/pacelab/ja/training-pace-calculator/)

イージーラン、マラソンペース、閾値走、インターバルの推奨トレーニングペースを確認できます。

### [心拍ゾーン](/pacelab/ja/heart-rate-zone-calculator/)

年齢、最大心拍数、安静時心拍数、乳酸閾値心拍数をもとに心拍ゾーンを計算します。

### [レース予想タイム](/pacelab/ja/race-time-predictor/)

最近の記録から、記録ベースの予測とVDOT予測でレースタイムを予測します。

### [エナジージェル計算](/pacelab/ja/energy-gel-calculator/)

レース距離、ペース、目標タイムをもとに、推奨ジェル本数と補給タイミングを計画します。
