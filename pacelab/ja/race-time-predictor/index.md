# レース予想タイム

[Pace Lab ホームへ戻る](/pacelab/ja/)

最近の走力をもとに、5K、10K、ハーフマラソン、マラソンの予想タイムを確認できます。

このレース予想タイムは、記録ベースの予測やVDOT予測などの方法を使って、複数距離の完走タイムを見積もるのに役立ちます。

## What you can do

- 最近の記録からレースタイムを予測
- 複数距離のパフォーマンスを推定
- 記録ベースの予測とVDOT予測を利用
- 目標ごとの予想タイムを比較

## When to use it

こんなときに使えます:

- 10Kやハーフの記録からマラソン予想タイムを知りたいとき
- 最近の結果をもとに現実的なレース目標を立てたいとき
- 距離ごとの予想タイムを比べたいとき
- 次のレース戦略を考えたいとき

## Example

10Kを50分で走った場合:

- ハーフマラソン予想: 約 1:50
- マラソン予想: 約 3:50-4:00

現在の仕上がりを確認し、達成しやすい目標タイムを設定するのに便利です。
