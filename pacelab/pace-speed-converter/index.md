# 페이스-속도 변환

[Pace Lab 메인으로 돌아가기](/pacelab/)

도로 러닝, 러닝머신, 트랙 훈련을 위해 러닝 페이스를 시속으로, 또는 시속을 페이스로 변환해 보세요.

이 페이스-속도 변환 도구는 페이스(min/km 또는 min/mi)와 시속(km/h 또는 mph) 사이를 쉽게 바꿔 볼 수 있어 러닝머신 설정이나 훈련 강도 조정에 유용합니다.

## What you can do

- 러닝 페이스를 시속으로 변환(km/h 또는 mph)
- 시속을 해당 러닝 페이스로 변환
- 입력값을 바로 조정하고 결과 즉시 확인
- 도로 러닝, 러닝머신, 트랙 훈련에 활용

## When to use it

이럴 때 활용해 보세요:

- 야외 러닝 페이스에 맞는 러닝 속도를 러닝머신에서 설정하고 싶을 때
- 훈련 계획의 km/h 값을 페이스로 바꾸고 싶을 때
- 단위 차이(km와 mile)에 따른 페이스 변화를 이해하고 싶을 때
- 훈련 강도를 빠르게 조정하고 싶을 때

## Example

러닝 페이스가 5:00분/km라면:

- 시속: 12.0km/h

러닝 속도가 10km/h라면:

- 페이스: 6:00분/km

러닝머신 훈련과 야외 러닝의 기준을 맞추는 데 유용하게 활용할 수 있습니다.
