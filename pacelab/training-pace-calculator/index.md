# 훈련 페이스 계산기

Easy Run, Marathon Pace, Tempo (LT), Interval (VO2)에 맞는 권장 훈련 페이스를 계산해 보세요.

이 훈련 페이스 계산기는 최근 대회 기록이나 목표 수행 수준을 바탕으로 훈련 종류별 적절한 강도를 설정하는 데 도움을 줍니다.

## What you can do

- 여러 훈련 유형에 대한 훈련 페이스 계산
- Easy Run, Marathon Pace, Tempo (LT), Interval (VO2) 권장 페이스 확인
- 최근 대회 기록을 기준으로 훈련 강도 설정
- 입력값을 바로 조정하고 훈련 페이스 범위 비교

## When to use it

이럴 때 활용해 보세요:

- 일상 훈련의 적정 페이스를 정하고 싶을 때
- Easy Run, Tempo (LT), 인터벌 훈련을 계획하고 싶을 때
- 이지 데이를 너무 빠르게 달리는 것을 피하고 싶을 때
- 더 체계적인 훈련 계획을 구성하고 싶을 때

## Example

최근 10K를 50분에 달렸다면:

- Easy Run 페이스는 더 여유 있고 편안하게 설정됩니다
- Tempo (LT) 페이스는 더 빠르지만 통제 가능한 강도로 설정됩니다
- Interval (VO2) 페이스는 더 빠르고 부담 있는 강도로 설정됩니다

훈련별 목적에 맞게 페이스를 정리하고, 더 일관된 리듬으로 훈련하는 데 활용할 수 있습니다.
