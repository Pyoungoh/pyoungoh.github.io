# 심박존 계산기

[Pace Lab 메인으로 돌아가기](/pacelab/)

나이, 최대 심박수, 안정 시 심박수, 역치 심박수를 기준으로 러닝 심박존 범위를 확인해 보세요.

이 심박존 계산기는 입력값에 따라 심박존 범위를 한 화면에서 확인하는 데 도움을 줍니다.

## 할 수 있는 것

- 입력값을 바탕으로 심박존 범위 확인
- 각 심박존의 범위를 한 화면에서 확인
- 값을 바꿨을 때 심박존 범위 비교

## 언제 사용하나요

이럴 때 활용해 보세요:

- 심박 기반 훈련 전에 목표 범위를 확인하고 싶을 때
- 쉬운 강도와 높은 강도의 심박존 범위를 함께 보고 싶을 때
- 입력값을 바꿨을 때 심박존 변화를 비교하고 싶을 때

## 예시

값을 입력하면:

- 낮은 강도부터 높은 강도까지의 심박존 범위를 확인할 수 있고
- 각 심박존을 한 목록에서 볼 수 있습니다

심박 기반 훈련 전에 범위를 점검할 때 활용할 수 있습니다.
