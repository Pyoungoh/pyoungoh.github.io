# 대회 기록 예측

[Pace Lab 메인으로 돌아가기](/pacelab/)

최근 기록을 바탕으로 여러 거리의 예상 기록을 확인해 보세요.

이 대회 기록 예측 도구는 기록 기반 예측과 VDOT 기반 예측 화면을 함께 확인하는 데 도움을 줍니다.

## 할 수 있는 것

- 최근 기록으로 예상 기록 확인
- 여러 거리의 예측 기록 비교
- 기록 기반 예측과 VDOT 기반 예측 함께 확인

## 언제 사용하나요

이럴 때 활용해 보세요:

- 최근 기록 하나를 기준으로 다른 거리 기록을 보고 싶을 때
- 거리별 예상 기록을 비교하고 싶을 때
- 목표 기록을 정하기 전에 참고 기록을 확인하고 싶을 때

## 예시

최근 기록을 입력하면:

- 여러 거리의 예상 기록을 볼 수 있고
- 기록 기반 예측과 VDOT 기반 예측을 비교할 수 있습니다

목표 기록을 정하기 전에 참고용 기록을 확인할 때 활용할 수 있습니다.
