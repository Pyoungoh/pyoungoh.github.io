# 훈련 페이스 계산기

[Pace Lab 메인으로 돌아가기](/pacelab/)

Easy Run, Marathon Pace, Tempo (LT), Interval (VO2)에 맞는 권장 훈련 페이스를 계산해 보세요.

이 훈련 페이스 계산기는 훈련 종류별 페이스 범위를 한 화면에서 확인하는 데 도움을 줍니다.

## 할 수 있는 것

- 훈련 유형별 권장 페이스 확인
- Easy Run, Marathon Pace, Tempo (LT), Interval (VO2)를 함께 확인
- 기록을 바꿨을 때 페이스 범위 비교

## 언제 사용하나요

이럴 때 활용해 보세요:

- 서로 다른 훈련일의 페이스를 정리하고 싶을 때
- 특정 훈련 전에 해당 페이스를 확인하고 싶을 때
- 최근 기록 기준 훈련 페이스 범위를 보고 싶을 때

## 예시

최근 기록을 입력하면:

- 훈련 유형별 페이스 범위를 따로 확인할 수 있고
- 각 훈련 유형을 한 목록에서 볼 수 있습니다

훈련별 페이스 기준을 정리할 때 활용할 수 있습니다.
